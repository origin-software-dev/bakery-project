Step 1 : Installation

This documentation will give you an understanding of how Bakery is structured and guide you in performing the most common functions

Note Please, read the whole section, before attempting to install the theme.

It is recommend to always backup your system before making any upgrades or changes!

Opencart Installtion

For installation of Opencart, you just need to follow these steps, Installation

Two ways to install Template:
1.1 Quickstart Install:

  1. Create new database
    2. Step 1: Open your Browser and type : You may access your database with the database management tool and perform the following steps...
    Step 2: Then create database
  2. Extract and Upload the whole Quickstart package on server
  3. Open browser and navigate to your site (where the quickstart uploaded) to load default Opencart installer with the database you just created.
  4. Enter the username and password for the administration
  5. Installation Complete

1.2 Template Install:
To install OpenCart template you need to upload these images folder on location images/catalog/
Upload all images on this images/catalog/ directory

"image" (Overwrite this folder)

To install OpenCart theme you need to follow this steps:

Step 1 : Go to Admin Extensions > Installer
Click On Upload Button. and Upload all "Themability Extensions".

After Upload All "Themability Extensions". And Install All "Themability Extensions".

Step 2 : Go to Admin Extensions > Extensions
Choose the extension type "Themes". and Install "Themability Theme".

Choose the extension type "Modules". and Install These 8 Modules.

Step 3(a) : Make sure Please Increase "$i so we can batch execute the queries" 100 to 10000
File Location: admin/controlller/tool/backup.php

Step 3(b) : Make sure Please replace 

<script type="text/javascript"><!--
$('textarea[data-oc-toggle=\'ckeditor\']').ckeditor({
	language:'{{ ckeditor }}'
});
//--></script>
to 
<script type="text/javascript"><!--
$('textarea[data-oc-toggle=\'ckeditor\']').ckeditor({
language:'{{ ckeditor }}',
allowedContent : true,
fillEmptyBlocks: false
});        
CKEDITOR.dtd.$removeEmpty['i'] = false;
//--></script>
File Location: extension/opencart/admin/view/template/module/html.twig

Go to Admin System > Maintenance > Backup & Restore
Click On Upload Button. and Select Upload Sql File "Bakery_Data_v4.0.x.sql".

Click On "Restore" Button.

Step 4 : Go to Admin Extensions > Extensions
Choose the extension type "Modules". and Install "Themability Megamenu" Module.

Choose the extension type "Themes". and Edit "Themability Theme".
